# Aprendamos Zapoteco del Sur

Sitio web oficial para presentar la aplicación de escritorio **Aprendamos Zapoteco del Sur**, además de proveer información de sus creadores en todas las etapas y los enlaces de descarga de la app.

# Comenzando 🚀

Estas instrucciones te permitirán obtener una copia del proyecto en funcionamiento en tu máquina local para propósitos de desarrollo y pruebas.

### Pre-requisitos 📋

- NodeJS versión 18.0.0 o superior
- NPM versión 8.0.0 o superior

### Instalación (desarrollo) 🔧

Instale las dependencias. Las dependencias ya se encuentran implícitas en el package del proyecto, solo basta ejecutar la siguiente instrucción:

```bash
npm i
```

Para mostrar las vistas del servidor local en modo desarrollo, levanta el servicio con el siguiente comando:

```bash
npm run dev
```

## Compilación y Despliegue 📦

Para la compilación y generación de los archivos estáticos listos para producción, ejecuta el siguiente script:

```bash
npm run build
```

## Construido con 🛠️

- ReactJS
- Vite
- Bootstrap 5
- Styled Components

Al ser una aplicación web, se implementaron las interfaces visuales mediante páginas web, lo que significa que se aplicaron las tecnologías HTML, CSS y JavaScript.
